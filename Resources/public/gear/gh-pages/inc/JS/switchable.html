Available data attributes

  • data-switchable = switchable transition (none, fade or slide) (value optionnal)
  • data-show = jQuery selector of the switchable html element to show
  • data-hide = jQuery selector of the switchable html element to hide
  • data-toggle = jQuery selector of the switchable html element to toggle

Using jQuery

// init with default options $("#myElement-1").switchable(); // call a method (open) : $("#myElement-1").switchable('toggle'); // init with default options and call immediately a method $("#myElement-2").switchable('hide'); // init with custom arguments $("#myElement-3").switchable( args );

Arguments

var args = { transitionIn : "show", transitionOut : "hide", transition : "none", time : 0, init : true } $("#myElement").switchable( args );

Methods

  • show() : show the switchable element
  • hide() : hide the switchable element
  • toggle() : toggle the element
