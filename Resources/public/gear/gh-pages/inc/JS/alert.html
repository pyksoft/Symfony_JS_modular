Available data attributes

  • data-alert = switchable transition (none, fade or slide) (optionnal)
  • data-open = jQuery selector of the alert html element to open
  • data-close = jQuery selector of the alert html element to close
  • data-close-this = no value. It will close the parent alert.

Using jQuery

Usage

// init with default options $("#myElement-1").alert(); // call a method (open) : $("#myElement-1").alert('open'); // init with default options and call immediately a method $("#myElement-2").alert('open'); // init with custom arguments $("#myElement-3").alert( args );

Arguments

var args = { show : false, overlay_color : "rgba(0,0,0,.7)", switchable : {}, zIndex : 5000, width : "400px", onClose : "", onOpen : "" } $("#myElement").alert( args );
